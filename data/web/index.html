Firmware Update (OTA)

Upload nextube-fw-ota.bin for over-the-air updates. Do not upload nextube-fw-full.bin — that is the full flash image for USB flashing only.

Web UI Update (SPIFFS)

Upload spiffs.bin to update the web interface and built-in assets without a USB flash. Required when the release notes mention web UI changes.

⚠️ Warning: this overwrites all SPIFFS flash — including any custom themes or images you have uploaded to the device. Built-in themes will be restored from the new image. Back up custom files first using the SPIFFS Files list below.
